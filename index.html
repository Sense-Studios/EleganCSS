    0. Configuration

    EleganCSS is what you get when you make a bad pun while playing with oklch colors -- it is also a drop-in solution for your CSS needs that requires exactly one configuration. Simply throw in some colors and fonts you like. Stop fiddling around with divs and spans; just write markup and EleganCSS will do the rest.

    You start with a simple configuration; color and fonts. Then add Elegance, and if you really need to, you can maybe also write some extra little custom styles.

    The need for EleganCSS.

    don't forget to add a catching quote

    I love Bootstrap. I have been using bootstrap since it infancy, and I think I have dreamt in bootstrap on more then one occasion. However, the world has moved on, since the days of margin: auto and float: left. I don't think I have to tell anyone that With the introduction of Flexbox and Grid, CSS has changed. There is no more need for a rigid 12-colomn grid anymore. We are not Sumerians, we can have seven columns if we really want. Or 13.
    We still have a need for styling though, and who can keep up with all the innovation the W3C has been putting out there the last five years. There has been native functions added to CSS monthly and browsers have been picking them up with great speed. There are only few of us who really want to read all the "five things you didn't know CSS could do" and "Are you still writing CSS as if it was 2015". They are pedantic. Still, wouldn't it be nice to have a modern CSS file that you can drop in, and does almost anything that bootstrap does, without javascript.
    That is where ElegenCSS comes in. It is a drop-in css that takes care of almost all your CSS needs with a single configuration.
    It takes what was good from bootstrap, but translates it to modern HTML standards and practices. That means, that if you write your html well, your styling will follow. No need for nested divs to set up that pesky horizontal form. Just add the correct class to the fieldset, section or form itself and it works. Straight out of the box and without hassle.
    It doesn't need to be backward compatible, because there is no backlog. You don't need to learn color theorie, because with oklhc all your color needs are taken care of and we simply calculate all derative colors you might require.
